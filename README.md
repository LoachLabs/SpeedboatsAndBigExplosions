# SBBE Music

The sexy website at [sbbemusic.com](http://sbbemusic.com).


## Dependencies

* [npm](https://npmjs.org)


## Build

``` bash
$ npm install
$ npm test
```

## Develop

``` bash
$ npm start
```

## Deploy

``` bash
$ npm run deploy
```
